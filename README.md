# Tic Tac Toe Game 🎮

A classic Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features ✨
- Two player gameplay
- Winner detection
- Draw detection
- Reset game option
- New game option
- Responsive design
- Smooth animations

## Technologies Used 💻
- HTML
- CSS
- JavaScript

## How to Run 🚀
1. Clone this repository
2. Open `index.html` in your browser
3. Start playing!
